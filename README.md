# Autumn Breakout
A autumn-themed Breakout clone built in Godot 4.5 as a beginner-friendly example project.
It is quite barebones, only featuring the paddle, ball, and bricks. The brick color is chosen at random from a pool.

Coding time, tracked with WakaTime (excluding art) (including physics debugging): 1 hour and 20 minutes

## Controls
- Move paddle: Left / Right Arrow

## Requirements
- Developed and tested in Godot 4.5
- Other Godot 4.x versions may work but are untested

## How to build / modify:
- Download the repository
- Import the project folder in Godot.
- No additional export settings are required.
- Use project > export in Godot to build.
